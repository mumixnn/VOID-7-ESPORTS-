# VOID 7 Esports

**VOID 7 Esports** is a professional esports organisation from **Kashmir, Srinagar (India)**.

## About Us
VOID 7 Esports was founded with the vision of representing Kashmiri talent on the national and global esports stage.  
We focus on competitive excellence, discipline, and long-term growth in the esports ecosystem.
